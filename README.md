#  Alke Wallet

Proyecto Front-end de una billetera digital en desarrollo como parte del **Módulo 2: Fundamentos del desarrollo Front-end**.

La aplicación simula una wallet digital que permite a los usuarios iniciar sesión, visualizar su saldo, realizar depósitos, enviar dinero y revisar el historial de transacciones.

---

##  Características principales

- Inicio de sesión de usuario
- Visualización de saldo disponible
- Depósitos de dinero
- Envío de dinero a contactos
- Historial de transacciones
- Interfaz responsive

*Proyecto actualmente en desarrollo*
